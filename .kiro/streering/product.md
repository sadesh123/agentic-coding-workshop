# Product: kiro-workshop-resume

A static, single-page resume website built with Astro, designed as a teaching boilerplate for a Kiro agentic coding workshop.

## Purpose

Students receive this pre-built site and use Kiro to customize it — editing content, styling, and structure — while learning agentic coding concepts. Instructors deploy it to Cloudflare Pages so students can see their resume live at the end of the workshop.

## Key Goals

1. **Pedagogical clarity** — easy to navigate for someone new to Astro and agentic coding
2. **Realistic starting point** — polished, complete resume with fictional placeholder content (persona: "Alex Rivera")
3. **Deployment simplicity** — zero-config static output deployable to Cloudflare Pages

## Hidden Learning Features

The codebase contains at least 6 "Easter Eggs" — hidden Kiro learning tips embedded as code comments (`// 🥚 KIRO TIP:` / `<!-- 🥚 KIRO TIP: -->`) and one hidden hover UI element (`<span class="kiro-tip" aria-hidden="true">`). An instructor reference guide lives at `EASTER_EGGS.md` in the project root.
